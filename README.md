# vue-run

## 当前项目是记录最佳实践 vue项目 ##

使用了一些vue的最佳实践：
1. 引入svg-sprite-loader，组件化引入svg图标
2. 增加路由权限控制
3. 增加按钮权限控制
4. 自动生成导航菜单
5. 数据交互：本地mock/服务器api/线上mock

## Project setup
```
npm install
```
